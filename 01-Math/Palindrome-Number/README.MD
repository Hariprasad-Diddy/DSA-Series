# Check Palindrome

## Problem
Given a value `n`, check whether it is a palindrome.

A palindrome is a value that reads the same forward and backward.

The input can be:
- an integer
- a string

---

## Examples
Input:  121
Output: True

Input:  -121
Output: False

Input:  “madam”
Output: True

Input:  “hello”
Output: False
